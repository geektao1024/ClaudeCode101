# 全局提示词设置

我强烈建议你设置这个全局提示词。很多人用Claude Code写出垃圾代码，就是因为没设置这个，让Claude一味地执行一些没意义的任务。

## 推荐提示词

```
用中文回答我
每次都用审视的目光，仔细看我输入的潜在问题，你要指出我的问题，并给出明显在我思考框架之外的建议
如果你觉得我说的太离谱了，你就骂回来，帮我瞬间清醒
```

## 设置方法

1. 命令行输入 `claude`
2. 输入 `/memory`
3. 选择 `User memory`

## 常见错误

最典型的错误就是直接让Claude写整个系统。比如"给我写个电商系统"，这种需求太笼统了，Claude写出来的代码基本没法用。正确的做法是把需求拆得很细：

❌ "给我写个完整的电商系统"  
✅ "帮我设计用户登录的数据库表结构"

❌ "帮我做个社交软件"  
✅ "这个朋友圈功能应该如何实现？先分析一下思路"

## 新手提问技巧

如果你是新手，不懂技术概念，直接告诉Claude你的背景，让它用生活例子解释。比如"我是做销售的，完全不懂编程，请用生活中的例子解释什么是数据库"。Claude很擅长这种类比解释，比那些技术文档好懂多了。

关键是要有耐心，不断追问到完全理解为止。
