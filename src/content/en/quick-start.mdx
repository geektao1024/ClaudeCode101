Welcome to Claude Code! This is a powerful AI programming assistant that helps you write, understand, and optimize code efficiently.

## Install Claude Code

### Method 1: Using npm (Recommended)

```bash
# Global installation
npm install -g claude-code

# Or using pnpm
pnpm add -g claude-code

# Or using yarn
yarn global add claude-code
```

### Method 2: Using Homebrew (macOS)

```bash
brew install claude-code
```

### Method 3: Using Scoop (Windows)

```bash
scoop install claude-code
```

## Configure API Key

<Callout type="info">
  **Get API Key**: Visit [Claude.ai](https://claude.ai) to create an account and get your API key
</Callout>

### Set Environment Variable

```bash
export ANTHROPIC_API_KEY="your_api_key_here"
```

### Or Use Configuration Command

```bash
claude config set api-key your_api_key_here
```

## Your First Project

### 1. Initialize Project

```bash
# Create new project directory
mkdir my-claude-project
cd my-claude-project

# Initialize Claude Code configuration
claude init
```

## Basic Usage

### Natural Language Commands

```bash
# Code generation
claude "Create a user login form"

# Code explanation
claude "Explain what this function does" -f utils.js

# Code optimization
claude "Optimize this algorithm's performance" -f algorithm.py

# Bug fixing
claude "Fix this bug" -f broken-code.js
```

### Project-level Operations

```bash
# Analyze entire project
claude "Analyze project architecture and provide improvement suggestions"

# Add features
claude "Add user authentication to this project"

# Refactor code
claude "Refactor this module for better maintainability"
```

## Advanced Features

### MCP (Model Context Protocol) Integration

```bash
# Set up MCP server
claude mcp add

# List available MCP servers
claude mcp list

# Configure project-level MCP
claude mcp config
```

### Custom Commands

Create custom commands in the `.claude/commands/` directory:

```markdown
// .claude/commands/deploy.md
Deploy current project to production environment, including build, test, and deployment steps
```

Use custom commands:

```bash
claude /deploy
```

### Hooks System

Configure automated workflows:

```json
// .claude/settings.json
{
  "hooks": {
    "pre-commit": "claude review --fix",
    "post-commit": "claude docs update"
  }
}
```

## Troubleshooting

### Common Issues

<Callout type="error">
  **API Key Error**: Check environment variables or use `claude config show` to view current configuration
</Callout>

