## 🎉 2024-2025 重大更新

### ✨ 正式发布

Claude Code 现已正式发布！经过广泛的研究预览和用户反馈，我们带来了:

- 📈 **扩展的开发者协作功能**
- 🔧 **GitHub Actions 后台任务支持**
- 🎨 **VS Code 和 JetBrains 原生集成**
- 🔄 **文件内直接编辑显示**

### 🛠️ 新增 API 功能

四个强大的新 API 功能，让开发者构建更强大的 AI 智能体：

1. **🧮 代码执行工具** - 直接在对话中运行代码
2. **🔗 MCP 连接器** - 模型上下文协议集成
3. **📁 Files API** - 文件管理和操作
4. **💾 提示缓存** - 高达1小时的提示缓存

## 🔥 核心功能更新

### 🎯 MCP (模型上下文协议) 集成

```bash
# 交互式 MCP 设置向导
claude mcp add

# 项目级 MCP 配置
# 在 .mcp.json 中配置并提交到仓库
```

**新功能：**
- 🏗️ 项目级 MCP 作用域
- 🔧 交互式设置向导
- 📥 从 Claude Desktop 导入 MCP 服务器
- ⏱️ 可配置启动超时（MCP_TIMEOUT）

### ⚡ 自定义斜杠命令

在 `.claude/commands/` 目录下的 Markdown 文件现在会自动显示为自定义斜杠命令：

```markdown
# .claude/commands/deploy.md
部署应用程序到生产环境
```

使用：`/deploy`

## 🚀 开始使用

准备好体验最新的 Claude Code 功能了吗？

### 💾 安装最新版本

```bash
# 全局安装
npm install -g claude-code

# 或使用 homebrew (macOS)
brew install claude-code

# 验证安装
claude --version
```

### 🎯 快速开始

1. **初始化项目**
   ```bash
   claude init
   ```

2. **配置设置**
   ```bash
   claude config
   ```

3. **开始编程**
   ```bash
   claude "帮我创建一个React组件"
   ```
