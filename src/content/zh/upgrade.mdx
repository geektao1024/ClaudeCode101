## 安装

### 安装最新版本

```bash
# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 验证安装
claude --version

# 更新到最新版本
claude update
```

### 版本管理

```bash
# 检查当前版本
claude --version

# 手动更新
npm install -g @anthropic-ai/claude-code@latest

# 运行诊断
claude doctor
```
