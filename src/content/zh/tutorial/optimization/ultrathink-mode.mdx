# Ultrathink 模式

在提示词里加上`ultrathink`关键字，让Claude进行深度思考。质量更高，但消耗更多token。

## 如何使用

在任意位置加上`ultrathink`关键字即可：

![Ultrathink 开启方式](/img/media/image7.png)

## 最佳组合

**Opus + Ultrathink + Plan Mode**是最强组合。思考质量高，能一次性解决复杂问题，避免反复修改。虽然单次消耗的token多，但因为不用反复兜圈子，总体上可能更省token。

## 适用场景

**适合用的场景**：
- 复杂架构设计
- 技术方案选择
- 复杂业务逻辑
- 算法优化

**不适合的场景**：
- 简单CRUD操作
- 样式调整
- Bug快速修复

## 实际效果

比如你让Claude设计电商系统架构，加上ultrathink后，它会深度分析业务需求，对比多种技术方案，考虑扩展性和维护性，给出非常详细的实施建议。不加ultrathink的话，可能只给你一个简单的方案，很多细节都没考虑到。

配合Plan Mode使用效果最好：先用ultrathink深度思考方案，确认无误后再让它具体实施。这样既保证了方案质量，又避免了后期返工。
