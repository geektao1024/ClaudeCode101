## 🎉 2024-2025 Major Updates

### ✨ General Availability

Claude Code is now generally available! After extensive research preview and user feedback, we bring you:

- 📈 **Extended developer collaboration features**
- 🔧 **GitHub Actions background task support**
- 🎨 **Native VS Code and JetBrains integration**
- 🔄 **Direct file editing display**

### 🛠️ New API Capabilities

Four powerful new API capabilities enable developers to build more powerful AI agents:

1. **🧮 Code Execution Tool** - Run code directly in conversations
2. **🔗 MCP Connector** - Model Context Protocol integration
3. **📁 Files API** - File management and operations
4. **💾 Prompt Caching** - Up to 1 hour of prompt caching

## 🔥 Core Feature Updates

### 🎯 MCP (Model Context Protocol) Integration

```bash
# Interactive MCP setup wizard
claude mcp add

# Project-level MCP configuration
# Configure in .mcp.json and commit to repository
```

**New Features:**
- 🏗️ Project-level MCP scope
- 🔧 Interactive setup wizard
- 📥 Import MCP servers from Claude Desktop
- ⏱️ Configurable startup timeout (MCP_TIMEOUT)

### ⚡ Custom Slash Commands

Markdown files in `.claude/commands/` directories now appear as custom slash commands:

```markdown
# .claude/commands/deploy.md
Deploy application to production environment
```

Usage: `/deploy`

### 🎣 Hooks System

A new hooks system allows you to create custom automation and workflow integrations.

### 📋 New Commands

## 🚀 Getting Started

Ready to experience the latest Claude Code features?

### 💾 Install Latest Version

```bash
# Global installation
npm install -g claude-code

# Or using homebrew (macOS)
brew install claude-code

# Verify installation
claude --version
```

### 🎯 Quick Start

1. **Initialize Project**
   ```bash
   claude init
   ```

2. **Configure Settings**
   ```bash
   claude config
   ```

3. **Start Coding**
   ```bash
   claude "Help me create a React component"
   ```
